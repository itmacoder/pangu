# pangu-core



## 使用方式

````
					 <dependency>
                <groupId>com.miaosuan.pangu</groupId>
                <artifactId>core-common</artifactId>
                <version>${pangu.core.version}</version>
            </dependency>

            <dependency>
                <groupId>com.miaosuan.pangu</groupId>
                <artifactId>core-orm</artifactId>
                <version>${pangu.core.version}</version>
            </dependency>

            <dependency>
                <groupId>com.miaosuan.pangu</groupId>
                <artifactId>core-cache</artifactId>
                <version>${pangu.core.version}</version>
            </dependency>

            <dependency>
                <groupId>com.miaosuan.pangu</groupId>
                <artifactId>core-feign</artifactId>
                <version>${pangu.core.version}</version>
            </dependency>

            <dependency>
                <groupId>com.miaosuan.pangu</groupId>
                <artifactId>core-mvc</artifactId>
                <version>${pangu.core.version}</version>
            </dependency>

            <dependency>
                <groupId>com.miaosuan.pangu</groupId>
                <artifactId>core-test</artifactId>
                <version>${pangu.core.version}</version>
            </dependency>
````
